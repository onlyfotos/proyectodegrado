Desarrollo del Proyecto

Para el desarrollo del proyecto se implementó una metodología de prototipo que consiste en la elaboración de un prototipo funcional y a pequeña escala del resultado final del proyecto.

Se utilizaron diversos componentes electrónicos como una protoboard, un Arduino UNO, un sensor de proximidad, un sensor de movimiento, dos diodos LED (verde y rojo), dos resistencias, un servomotor, cables macho-macho y macho-hembra, y una pantalla LCD 16x2 I2C.

Diseño y Complicaciones

Para el diseño del prototipo se utilizó la plataforma de simulación Tinkercad, donde se realizó el código y las conexiones que tendría el prototipo.

Se realizaron diversas pruebas en la institución en el área del laboratorio de computación, donde se encontraron problemas de lógica en el código y de conexiones en el prototipo.

Estos problemas se resolvieron revisando y corrigiendo el código con ayuda de una inteligencia artificial que indicaba dónde se encontraban las fallas de lógica para poder ser corregidas manualmente, así como la corrección de conexiones físicas con ayuda del tutor de grado.
